# 밥괘 브랜드 가이드

## 현재 이름

제품의 현재 작업명은 **밥괘**다.

`밥`과 `점괘`를 결합한 이름으로, 매일 오전 11시에 그날의 점심 후보를 재미있는 점괘처럼 알려준다는 의미를 담는다. 현재 제품과 메시지 설계에서 공식적으로 사용하되, 실제 사용 경험에 따라 변경을 검토할 수 있다.

## 핵심 콘셉트

- 매일 오전 11시에 오늘의 점심 운세를 공개한다.
- 사내 구성원이 함께 만든 추천 목록을 바탕으로 밥괘를 뽑는다.
- 밥괘는 결정을 강요하는 정답이 아니라 대화를 시작하기 위한 재미있는 후보이다.
- 예언과 점괘의 분위기를 활용하되 특정 참고 사례의 이름이나 캐릭터를 그대로 모방하지 않는다.
- 기능이 발전하더라도 유머와 이해하기 쉬운 표현을 함께 유지한다.

## 기본 문구

대표 문구는 다음과 같다.

```text
오늘의 밥괘가 나왔습니다.
```

제품을 짧게 설명할 때는 다음 표현을 우선 사용한다.

```text
매일 오전 11시에 공개되는 오늘의 점심 운세
```

초기 메시지의 개념 예시는 다음과 같다. 실제 메시지 구성은 별도로 합의한 뒤 확정한다.

```text
🔮 오늘의 밥괘가 나왔습니다.

1. 마루식당 — 제육볶음
2. 소바집 — 냉소바
3. 대성집 — 순댓국

점괘를 믿을지는 여러분의 몫입니다.
```

## 말투 원칙

- 친근하고 장난스럽되 업무 채널에서 부담스럽지 않게 짧게 쓴다.
- 점괘, 기운, 운명 같은 표현은 양념처럼 사용하고 모든 문장을 과하게 콘셉트화하지 않는다.
- 특정 식당을 반드시 선택해야 하는 것처럼 말하지 않는다.
- 구성원의 선택이나 취향을 평가하지 않는다.
- 같은 농담을 매일 길게 반복하지 않는다.
- 가격, 메뉴, 링크 등 실제 선택에 필요한 정보는 콘셉트보다 명확하게 전달한다.
- 후보 선택에 사용하는 `1️⃣`, `2️⃣`, `3️⃣` 반응은 장식이 아니라 기능적 입력이므로 제목의 `🔮` 이모지 제한과 별도로 취급한다.

## 아직 결정하지 않은 사항

- Slack 앱 아이콘과 대표 색상
- 밥괘를 의인화할지와 구체적인 캐릭터 설정
- 메시지마다 문구를 바꿀지 또는 일관된 문구를 유지할지
- 추천 후보별 표현과 이모지 사용 방식
- 이모지 반응 또는 스레드를 유도하는 문구

이 항목들은 구현 전에 선택지와 트레이드오프를 검토하고 확정한다.
